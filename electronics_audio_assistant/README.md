# Audio Electronics Assistant

This simple toolkit provides calculators and utilities for common audio-related electronics tasks.

## Features

- **555 Timer** frequency and duty cycle calculator.
- **40106 Oscillator** frequency calculator.
- **LED Resistor** calculator.
- **Op-Amp Gain** calculator for a non-inverting amplifier.
- **Transistor Bias** simple bias network helper.
- **Datasheet Search** opens a web browser for the part number.
- **Oscilloscope** capture from default audio input and display waveform.

Run with:

```bash
python3 assistant.py
```

Requires `numpy`, `matplotlib` and `sounddevice` packages.
